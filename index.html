Data Visualization & Analytics

Vehicle Market
Insights Dashboard

A comprehensive analytics tool designed to bring transparency to the used car market. By processing over 47,000 data points, this dashboard empowers users to identify fair market values and depreciation trends instantly.

The Technical Challenge

Raw vehicle data is often noisy and unstructured. The goal was to clean, normalize, and visualize this high-volume dataset in a way that allows for real-time filtering without performance lag.

Key Features & Tools

  • Python & Pandas: Used for robust ETL (Extract, Transform, Load) processes to clean inconsistent mileage and price data.
  • Plotly Express: Selected for its interactive capabilities, allowing users to zoom and hover over scatter plots to detect outliers.
  • Streamlit: Deployed for rapid prototyping, turning Python scripts into a deployable web app in record time.
